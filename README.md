Annona core

## 简介
Telegram 群聊关键词提醒

支持跨服务器部署。

## 组成
### annona_core 

* 授权管理
* API设计
* DB2REDIS

### annona_client 

信息采集userbot

### annona_bot

消息通知机器人

## 配置运行

配置模版在 `_default_env` 文件夹
